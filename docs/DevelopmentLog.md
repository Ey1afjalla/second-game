# 开发日志

## 2026-07-13 v0.1.0 Web MVP

完成内容：

1. 初始化 React + TypeScript + Vite Web 工程。
2. 实现《夜市怪谈摊主》的三天经营 MVP。
3. 实现基础数值：铜钱、声望、满意度、库存、天数。
4. 实现基础循环：补货、接待 NPC、销售、收摊。
5. 实现三类以上 NPC 和五种商品。
6. 实现事件埋点，并在运营看板展示接待、收入、转化和流失。
7. 实现本地模拟 AI 运营日报，输出证据、风险、建议和活动配置草案。
8. 实现配置视图，展示第一版 JSON 配置结构。

技术选择：

1. 第一阶段使用纯前端状态承载游戏和运营数据，降低启动成本，方便面试演示。
2. UI 使用手写 CSS，保证能快速控制视觉层级和响应式效果。
3. 使用 lucide-react 提供工具类图标，提升后台工具感。

问题与解决：

1. PowerShell 脚本策略阻止 `npm` 执行，改用 `npm.cmd`。
2. Vite 交互式初始化因目录非空取消，改为手工创建项目文件。
3. TypeScript 默认解析配置不适配 Vite 7，改为 `moduleResolution: bundler` 并补充 Node 类型。

版本状态：

1. 已通过 `npm run build`。
2. 已完成浏览器验证：首屏渲染、接待顾客、运营看板数据更新、控制台错误检查、390px 窄屏溢出检查。
3. 本版本完成后需要推送到 GitHub。
