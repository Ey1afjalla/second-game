# 夜市怪谈摊主

面向“游戏运营 / AI Agent 工程师”岗位的 AI LiveOps 作品集项目。

第一版已经完成 Web MVP：

1. 三天夜市经营循环。
2. 进货、接待 NPC、销售和收摊结算。
3. 基础事件埋点。
4. 运营数据看板。
5. 本地模拟 AI 运营日报和活动配置草案。
6. 配置驱动视图，后续接入 schema 校验和回滚。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
